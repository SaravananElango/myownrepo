How to create WIP:

Create a WIP with ID name 
Create a Provider with ID name and OIDC connection and give github OIDC url
Set default attributes as 
google.subject	assertion.sub
attribute.repository_owner	assertion.repository_owner
attribute.repository	assertion.repository

Also mention the repository name 

assertion.repository == 'repo_name'

Now grant access to service account which is gonna impersonate as WIP

Build WIP path for creating github secret as projects/'$project_number'/locations/global/workloadIdentityPools/'$pool_name'/providers/$provider_name

Above path has provider id, project number, wip id ... build it based on your id's
